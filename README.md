#puck


## 开发环境
React + Redux + Express
使用 webpack 打包组件，gulp 构建

### Sublime 开发必备插件: 
EditorConfig: 配合 .editorconfig 文件自动设置格式
Babel: ES6/JSX 语法高亮
HTML-CSS-JS Pretify: 配合 .jsbeautifyrc 文件对JSX文件格式化
其他有 LESS ReactJs Emmet SideBarEnhancements SublimeLinter-jsxhint Gulp Git 等
